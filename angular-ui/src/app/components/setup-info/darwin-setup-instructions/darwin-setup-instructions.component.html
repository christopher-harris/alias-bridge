Activate Your Aliases (One-Time Setup)

To make aliases managed here available in your terminal, you need to tell your shell to load them. Add the following lines, preferably at the end, of your shell's main configuration file:

  
{{aliasConnectorText}}

Common Configuration Files:

  • Zsh: ~/.zshrc
  • Bash: ~/.bashrc (most Linux, WSL, Git Bash) or ~/.bash_profile (some macOS setups)

After adding the lines, you must open a new terminal session for the changes to take effect.
